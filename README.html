Useful Git commands:

Description Command
Cloning from github
[Open the directory] cd c:/xampp/htdocs/hello-world
[Print working directory] pwd
[List all files] ls -al
[Cloning] git clone https://github.com/EstebanBojorquez/hello-world.git
[Status of edited files] git status
[Commit -a (all) -m (Message)] git commit -a -m "Message"
[Listing configuration settings] git config --list
[Username] git config --global user.name "Your name"
[Email] git config --global user.email your@example.com
[Log #commits] git log -2
[Escape VIM] :q
[Remotes associated with project] git remote -v
[Push (remote) (branch)] git push origin master
Initializing from local
[Initialize a repo] git init
[Add a file to the staging area - Previous to commit] git add test.txt
[Add everything] git add .
[Add a remote] git remote add origin https://github.com/EstebanBojorquez/hello-world.git
[Pull newest copy from github] git pull origin master
Github pages
[Display content on live url] change default master branch to "gh-pages"
[Live url] https://estebanbojorquez.github.io/hello-world/index.html
